# Example Skill

Cette compétence sert d'exemple pour vous montrer comment organiser vos instructions personnalisées.

## Utilisation
Utilisez cette structure pour guider l'agent (Antigravity) sur des tâches répétitives ou complexes propres à votre projet.

### Structure attendue :
- **Frontmatter YAML** : `name` et `description` (obligatoire).
- **Titre principal** : Nom de la compétence.
- **Détails** : Instructions, bonnes pratiques et exemples.
